# 公益财务相关法律法规总览

> 本文件汇总公益机构财务管理涉及的核心法律法规，按层级和主题分类。
> 供 AI 引用法条时快速定位，也供一线人员了解"我们要遵守哪些法"。

---

## 一、上位法（人大立法）

| 法规名称 | 文号/时间 | 核心内容 | 与财务的关联 |
|---------|---------|---------|------------|
| **《中华人民共和国慈善法》** | 2023年修订 | 慈善活动的基本法 | 捐赠收入管理、信息公开、支出比例、关联交易 |
| **《中华人民共和国公益事业捐赠法》** | 1999年 | 公益捐赠行为的基本法 | 捐赠行为定义、票据开具依据 |
| **《中华人民共和国会计法》** | 2024年修正 | 会计核算的基本法 | 记账规范、凭证管理、会计责任 |
| **《中华人民共和国企业所得税法》** | 2018年修正 | 企业所得税征收 | 公益捐赠税前扣除（第9条）、免税收入（第26条） |
| **《中华人民共和国个人所得税法》** | 2018年修正 | 个人所得税征收 | 个人公益捐赠扣除（第6条）、代扣代缴 |
| **《中华人民共和国增值税法》** | 2026年1月1日施行 | 增值税征收 | 服务收入的增值税处理 |

## 二、行政法规（国务院）

| 法规名称 | 文号 | 核心内容 |
|---------|------|---------|
| **《基金会管理条例》** | 国务院令第400号 | 基金会设立、运作、财务管理、监督 |
| **《现金管理暂行条例》** | 国务院令第12号 | 现金使用范围、结算起点（1000元） |
| **《企业所得税法实施条例》** | 国务院令第512号 | 公益捐赠扣除比例12%、免税条件细则 |

## 五、法规适用速查

当遇到具体问题时，快速定位适用法规：

| 问题场景 | 核心适用法规 |
|---------|------------|
| 捐赠票据怎么开 | 财综〔2024〕1号 |
| 支出比例算法 | 民发〔2016〕189号 |
| 能不能税前扣除 | 2020年第27号公告 + 企业所得税法第9条 |
| 免税资格怎么申请 | 财税〔2018〕13号 |
| 会计科目怎么设 | 民间非营利组织会计制度 |
| 信息公开要求 | 民政部令第81号（2025修订） |
| 年检/年报要求 | 慈善法 + 基金会管理条例 + 民政部年度通知 |
| 关联交易 | 慈善法第52条 + 慈善组织信息公开办法 |
| 现金管理 | 现金管理暂行条例 |
| 审计要求 | 基金会管理条例 + 财务审计通知 |
