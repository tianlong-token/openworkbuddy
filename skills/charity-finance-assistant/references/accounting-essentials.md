# 民间非营利组织会计制度 实操要点

> 来源：《民间非营利组织会计制度》（财会〔2004〕7号，2024年修订版）
> 本文件为会计制度核心内容的实操化摘要，面向公益机构一线财务人员。
> 完整制度请查阅财政部会计司官网。

---

## 一、适用范围

同时满足以下条件的民间非营利组织：
1. 不以营利为宗旨和目的
2. 资源提供者不因此享有该组织的所有权益
3. 资源提供者不取得经济回报

具体包括：社会团体、基金会、社会服务机构（民办非企业单位）等。

## 二、会计核算基础

| 项目 | 规定 |
|------|------|
| 记账基础 | **权责发生制** |
| 记账货币 | 人民币 |
| 会计期间 | 日历年度（1月1日至12月31日） |
| 记账方法 | 借贷记账法 |

> 💡 一线人员注意：权责发生制意味着"钱什么时候该记"看的是交易发生的时间，不是收到/付出钱的时间。
> 例：12月签了合同约定1月付款，费用应记在12月。

## 三、常用会计科目

### 3.1 资产类

| 科目 | 用途 | 一线人员理解 |
|------|------|------------|
| 现金 | 库存现金 | 保险箱里的现金 |
| 银行存款 | 银行账户余额 | 银行卡/对公账户里的钱 |
| 短期投资 | 1年内的投资 | 买的理财产品 |
| 应收账款 | 应收未收的款项 | 别人欠我们的钱 |
| 预付账款 | 预先支付的款项 | 我们先付了但还没收到东西/服务 |
| 存货 | 物资 | 仓库里的捐赠物资/办公用品 |
| 固定资产 | 长期使用的资产 | 电脑、车辆、办公家具 |
| 累计折旧 | 固定资产已消耗的价值 | 东西用旧了要扣的钱 |

### 3.2 负债类

| 科目 | 用途 | 一线人员理解 |
|------|------|------------|
| 应付账款 | 应付未付的款项 | 我们欠别人的钱 |
| 预收账款 | 预先收到的款项 | 别人先给了钱但我们还没提供服务 |
| 应付工资 | 应发未发的工资 | 这个月还没发的工资 |
| 应交税金 | 应缴未缴的税 | 要交的税 |

### 3.3 净资产类（最重要！）

| 科目 | 用途 | 一线人员理解 |
|------|------|------------|
| **限定性净资产** | 捐赠人指定了用途的净资产 | 指定了"只能用于XX项目"的钱 |
| **非限定性净资产** | 没有使用限制的净资产 | 机构可以自由支配的钱 |

### 3.4 收入类

| 科目 | 常见来源 |
|------|---------|
| 捐赠收入 | 个人捐赠、企业捐赠、平台筹款 |
| 会费收入 | 会员会费（社会团体） |
| 提供服务收入 | 培训收费、咨询服务 |
| 政府补助收入 | 财政拨款、购买服务 |
| 投资收益 | 理财收益、银行利息 |
| 其他收入 | 资产处置收益等 |

### 3.5 费用类

| 科目 | 常见内容 | 归入年报哪里 |
|------|---------|------------|
| 业务活动成本 | 项目直接支出 | 慈善活动支出 |
| 管理费用 | 行政办公支出 | 管理费用（有上限） |
| 筹资费用 | 筹款活动费用 | 筹资费用 |
| 其他费用 | 上述之外的费用 | 其他支出 |

## 四、限定性与非限定性的处理（核心）

### 4.1 收入确认

```
收到捐赠时：
  ├─ 捐赠人指定了用途/时间 → 确认为"限定性"收入
  │   例：张先生捐10万元，指定用于XX助学项目
  │   分录：借：银行存款 100,000
  │         贷：捐赠收入—限定性收入 100,000
  │
  └─ 捐赠人没有附加限制 → 确认为"非限定性"收入
      例：李女士捐5万元，没有说用途
      分录：借：银行存款 50,000
            贷：捐赠收入—非限定性收入 50,000
```

### 4.2 限定性收入使用完毕后的转换

```
当限定性收入被用于指定用途后：
  → 做"限定性净资产转为非限定性净资产"
  分录：借：限定性净资产
        贷：非限定性净资产
```

### 4.3 判断标准

| 情况 | 分类 |
|------|------|
| 捐赠协议中指定了项目名称 | 限定性 |
| 转账备注中注明了用途 | 限定性 |
| 平台筹款时选了特定项目 | 限定性 |
| 捐赠人口头说了用途但没有书面约定 | 建议按限定性处理并补书面确认 |
| 什么都没说 | 非限定性 |
| 指定了使用期限 | 在期限内为限定性，到期后转非限定性 |

## 五、财务报表

### 5.1 三张主表

1. **资产负债表**：反映某一日期的资产、负债和净资产
2. **业务活动表**：反映一定期间的收入、费用和净资产变动（类似企业的利润表）
3. **现金流量表**：反映一定期间的现金流入和流出

### 5.2 业务活动表结构（最常用）

```
业务活动表
编制单位：              年度：

                      非限定性    限定性    合计
一、收入
  捐赠收入
  政府补助收入
  提供服务收入
  投资收益
  其他收入
  收入合计

二、费用
  业务活动成本
  管理费用
  筹资费用
  其他费用
  费用合计

三、限定性净资产转为
    非限定性净资产

四、净资产变动额
```

## 七、关联方交易识别与披露

### 7.1 谁是"关联方"？

依据《民间非营利组织会计制度》和《慈善组织信息公开办法》，以下属于关联方：

| 序号 | 关联方类型 | 举例 |
|------|-----------|------|
| 1 | 该组织的发起人/主要捐赠人 | 出资设立基金会的企业 |
| 2 | 该组织的管理人员（理事、监事、秘书长等） | 理事长张三 |
| 3 | 管理人员的近亲属 | 理事长的配偶、子女、父母、兄弟姐妹 |
| 4 | 管理人员或其近亲属担任理事/高管的其他组织 | 理事长同时担任A公司董事 |
| 5 | 管理人员或其近亲属控制/共同控制的组织 | 秘书长的配偶开的公司 |
| 6 | 本组织的分支机构 | 基金会设立的地方办事处 |
| 7 | 与本组织受同一单位控制的其他组织 | 同一业务主管单位下的兄弟机构 |
| 8 | 管理人员或其近亲属设立的其他非营利组织 | 理事设立的另一个社会组织 |

> 💡 **简易判断法**：理事/监事/秘书长以及他们家人有关系的任何单位，都可能是关联方。拿不准的宁可先列上，让审计师来判断。

### 7.2 哪些交易需要披露？

与关联方之间发生的以下行为都需要披露：
- 捐赠（给关联方或从关联方接受）
- 资助（资助关联方的项目）
- 投资（投资关联方的产品/企业）
- 采购/服务（向关联方采购物资或购买服务）
- 资金往来（借款、垫付等）
- 租赁（租用关联方的场地或设备）

### 7.3 关联方名单报送

- **时间**：每年 **5月31日前** 向登记管理机关报送
- **内容**：关联方名称、与本组织的关系说明

### 7.4 关联方名单模板

```
[机构名称] 关联方名单
报送年度：____年

| 序号 | 关联方名称 | 关联方类型 | 与本组织关系 | 统一社会信用代码 |
|------|-----------|-----------|------------|---------------|
| 1 | 张三 | 个人 | 本组织理事长 | — |
| 2 | XX有限公司 | 企业 | 理事长张三持股60% | 91XXXXXXXXX |
| 3 | 李四 | 个人 | 理事长张三之配偶 | — |
| 4 | YY基金会 | 社会组织 | 理事王五同时任其理事 | 53XXXXXXXXX |
| ... | | | | |

编制人：          审核人：          日期：
```

### 7.5 关联交易情况说明模板

```
[机构名称] ____年度关联交易情况说明

一、关联交易汇总

| 序号 | 关联方 | 交易类型 | 交易内容 | 金额（元） | 定价依据 | 审批情况 |
|------|-------|---------|---------|----------|---------|---------|
| 1 | XX公司 | 采购 | 采购办公电脑10台 | 45,000 | 市场询价 | 理事会决议通过 |
| 2 | 张三 | 捐赠 | 接受理事长捐赠 | 100,000 | — | — |
| ... | | | | | | |

二、关联交易说明

1. 上述关联交易均已经理事会审议通过（会议日期：____，决议编号：____）
2. 涉及采购/服务类关联交易的定价依据：[说明是否参考市场价、是否有比价]
3. 关联方张三在涉及其利益的表决事项中回避表决

三、声明

本组织确认以上关联交易信息真实、完整，不存在未披露的关联交易。

法定代表人签字：              日期：
```

> ⚠️ **关键合规要求**：关联方在涉及其利益的理事会表决中须**回避**（依据《慈善法》第52条）。
