# 公益性捐赠税前扣除政策速查

> 来源：《关于公益性捐赠税前扣除有关事项的公告》（财政部 税务总局 民政部公告2020年第27号）等
> 本文件为政策核心内容摘要，供解答捐赠人咨询时参考。具体税务处理建议咨询专业税务师。

---

## 一、企业捐赠

| 项目 | 规定 |
|------|------|
| 扣除比例 | 年度利润总额的 **12%** 以内 |
| 超额处理 | 可结转以后 **3年** 内扣除 |
| 扣除条件 | 通过具有公益性捐赠税前扣除资格的组织 |
| 有效凭证 | 省级以上财政部门监（印）制的公益事业捐赠票据 |

### 特殊全额扣除
- 扶贫捐赠（有政策时效，需确认最新文件）
- 国务院规定的其他全额扣除情形

## 二、个人捐赠

| 项目 | 规定 |
|------|------|
| 扣除比例 | 应纳税所得额的 **30%** 以内 |
| 超额处理 | 一般 **不可结转** |
| 扣除条件 | 通过具有公益性捐赠税前扣除资格的组织 |
| 有效凭证 | 公益事业捐赠票据 |

### 特殊全额扣除
- 向教育、扶贫、济困等公益慈善事业捐赠（特定情形，需确认适用条件）

## 三、公益性社会组织税前扣除资格

### 3.1 资格认定
- 由财政部、税务总局、民政部联合确认并公布名单
- 省级及以下的由对应级别部门确认

### 3.2 资格有效期
- 税前扣除资格有效期一般为 **3年**（以联合公告为准，如"2022-2024年度"、"2023-2025年度"）

> ⚠️ 注意区分两个不同的资格：
> - **"非营利组织免税资格"**（财税〔2018〕13号）：有效期 **5年**，影响的是机构自身收入是否免企业所得税
> - **"公益性捐赠税前扣除资格"**（2020年第27号公告）：有效期 **3年**，影响的是捐赠人能否凭票据税前扣除
> 两者是不同的资格，需要分别申请。

### 3.3 查询方式
- 财政部/税务总局/民政部联合公告的名单
- 各省级财政/税务/民政部门公布的名单
- "慈善中国"平台可查询部分信息

### 3.4 机构如何确认自身是否有资格
1. 查看本机构是否在最近一期的公告名单中
2. 确认资格有效期是否覆盖当前年度
3. 如不确定，向登记管理机关或主管财政部门咨询

## 四、开具票据的注意事项

### 4.1 企业捐赠开票
- 必须填写企业全称和统一社会信用代码
- 金额必须与实际捐赠金额一致
- 建议在备注栏注明对应项目名称

### 4.2 个人捐赠开票
- 填写个人真实姓名
- 如个人需要，可在备注栏注明身份证号（需征得同意）
- 通过筹款平台的捐赠，由接收捐赠的机构开具

### 4.3 非货币捐赠开票
- 按公允价值开具
- 公允价值确定依据：有关凭据（发票、报关单等）、评估报告、市场价格
- 建议在备注栏注明物品名称、数量及定价依据

## 五、常见问答

### Q1：通过腾讯公益/支付宝公益平台捐赠，谁开票据？
由项目执行机构（接受捐赠的公益性社会组织）开具，不是平台开具。

### Q2：捐赠后多久能拿到票据？
法规没有规定具体时限，建议机构在收到捐赠后30日内开具。线上小额捐赠通常由机构统一开具电子票据。

### Q3：匿名捐赠能开票据吗？
可以开具，但捐赠人如需用于税前扣除，票据上需要有真实姓名/名称。

### Q4：配捐（企业/平台配捐）部分算不算捐赠人的捐赠？
不算。配捐是企业/平台另行捐出的款项，不属于捐赠人的捐赠额，不应开在捐赠人名下。

### Q5：机构没有税前扣除资格，能开捐赠票据吗？
可以开捐赠票据（证明捐赠事实），但该票据不能用于捐赠人的税前扣除。必须如实告知捐赠人。
