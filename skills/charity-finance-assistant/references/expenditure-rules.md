# 慈善组织年度支出和管理费用规定摘要

> 来源：《关于慈善组织开展慈善活动年度支出和管理费用的规定》（民发〔2016〕189号）
> 及《慈善法》（2023年修订）第六十条
> 施行日期：2016年10月11日
> ⚠️ 本文件经2026-04-09精准度复核，已与法规原文逐条比对校准。

---

## 一、法律依据

**《慈善法》第六十条**（原文）：

> 慈善组织应当积极开展慈善活动，充分、高效运用慈善财产，并遵循管理费用最必要原则，厉行节约，减少不必要的开支。慈善组织中具有公开募捐资格的基金会开展慈善活动的年度支出，不得低于上一年总收入的百分之七十或者前三年收入平均数额的百分之七十；年度管理费用不得超过当年总支出的百分之十，特殊情况下，年度管理费用难以符合前述规定的，应当报告其登记的民政部门并向社会公开说明情况。
>
> 具有公开募捐资格的基金会以外的慈善组织开展慈善活动的年度支出和管理费用的标准，由国务院民政部门会同国务院财政、税务等部门依照前款规定的原则制定。

具体比例标准由**民发〔2016〕189号**文件细化规定。

---

## 二、慈善活动支出（最低比例）

### 2.1 具有公开募捐资格的基金会（第七条）

- 年度慈善活动支出 ≥ 上年总收入的 **70%**

### 2.2 具有公开募捐资格的社会团体和社会服务机构（第七条）

- 年度慈善活动支出 ≥ 上年总收入的 **70%**

### 2.3 不具有公开募捐资格的基金会（第八条，按净资产规模4档）

| 上年末净资产 | 年度慈善活动支出下限 |
|---|---|
| ≥ 6000万元 | ≥ 上年末净资产的 **6%** |
| 800万元（含）～ 6000万元 | ≥ 上年末净资产的 **6%** |
| 400万元（含）～ 800万元 | ≥ 上年末净资产的 **7%** |
| < 400万元 | ≥ 上年末净资产的 **8%** |

### 2.4 不具有公开募捐资格的社会团体和社会服务机构（第九条，按净资产规模4档）

| 上年末净资产 | 年度慈善活动支出下限 |
|---|---|
| ≥ 1000万元 | ≥ 上年末净资产的 **6%** |
| 500万元（含）～ 1000万元 | ≥ 上年末净资产的 **7%** |
| 100万元（含）～ 500万元 | ≥ 上年末净资产的 **8%** |
| < 100万元 | ≥ 上年末净资产的 **8%**，且 ≥ 上年总收入的 **50%** |

> ⚠️ 注意最后一档（<100万净资产）有**双重标准**，需同时满足净资产8%和总收入50%。

---

## 三、管理费用（上限比例）

### 3.1 具有公开募捐资格的基金会（《慈善法》第60条 + 第七条）

- 管理费用 ≤ 当年总支出的 **10%**

### 3.2 具有公开募捐资格的社会团体和社会服务机构（第七条）

- 管理费用 ≤ 当年总支出的 **13%**

### 3.3 不具有公开募捐资格的基金会（第八条，按净资产规模4档）

| 上年末净资产 | 管理费用上限 |
|---|---|
| ≥ 6000万元 | ≤ 当年总支出的 **12%** |
| 800万元（含）～ 6000万元 | ≤ 当年总支出的 **13%** |
| 400万元（含）～ 800万元 | ≤ 当年总支出的 **15%** |
| < 400万元 | ≤ 当年总支出的 **20%** |

### 3.4 不具有公开募捐资格的社会团体和社会服务机构（第九条，按净资产规模4档）

| 上年末净资产 | 管理费用上限 |
|---|---|
| ≥ 1000万元 | ≤ 当年总支出的 **13%** |
| 500万元（含）～ 1000万元 | ≤ 当年总支出的 **14%** |
| 100万元（含）～ 500万元 | ≤ 当年总支出的 **15%** |
| < 100万元 | ≤ 当年总支出的 **20%** |

### 3.5 特别豁免（第十一条）

> **年度管理费用低于20万元人民币的慈善组织**，不受第七条、第八条、第九条管理费用比例的限制。

这条对小型公益机构非常重要——如果一年管理费用不到20万，即使占比超标也不违规。

---

## 四、慈善活动支出的定义（第四条）

以下支出属于"慈善活动支出"，在"业务活动成本"下核算：

1. 直接或委托其他组织资助给受益人的款物
2. 为提供慈善服务和实施慈善项目发生的**人员报酬、志愿者补贴和保险**，以及使用房屋、设备、物资发生的相关费用
3. 为管理慈善项目发生的**差旅、物流、交通、会议、培训、审计、评估**等费用

> 💡 注意第2和第3点：项目人员的工资、项目差旅、项目培训等都可以计入慈善活动支出，不一定要归到管理费用！

## 五、管理费用的定义（第五条）

以下属于管理费用：

1. 理事会等决策机构的工作经费
2. **行政管理人员**的工资、奖金、住房公积金、住房补贴、社会保障费
3. 办公费、水电费、邮电费、物业管理费、差旅费、折旧费、修理费、租赁费、无形资产摊销费、资产盘亏损失、资产减值损失、因预计负债所产生的损失、聘请中介机构费等

> 💡 关键区分：**项目人员的工资 → 慈善活动支出**，**行政管理人员的工资 → 管理费用**。项目差旅 → 慈善活动支出，行政差旅 → 管理费用。

## 六、共同费用的分摊（第六条）

某些费用如果属于慈善活动、其他业务活动、管理活动共同发生，且不能直接归属某一类活动的，应当将这些费用按照**合理的方法**在各项活动中进行分配。

常见分摊方法：
- 按人员比例分摊（如工资按项目人员/行政人员比例）
- 按面积比例分摊（如办公室租金按项目团队使用面积占比）
- 按工时比例分摊（如共用人员按投入各类活动的时间比例）

## 七、计算规则的补充说明（第十条）

1. 计算支出比例时，可以用**前三年收入/净资产的平均数**代替上年数据（取两者中对组织较有利的）
2. 上年总收入中应**扣除限定性收入**，再**加回**当年解除时间限定的净资产
3. 上年总收入含接受的**货币**捐赠和**非货币**捐赠

---

## 八、合规自查速算

### 公募基金会速算

```
慈善活动支出占比 = 本年慈善活动支出 ÷ 上年总收入 × 100%
  → 达标线：≥ 70%

管理费用占比 = 管理费用 ÷ 当年总支出 × 100%
  → 达标线：≤ 10%
```

### 非公募基金会速算

```
先查上年末净资产是多少，确定所在档位：
  ≥6000万 → 支出≥6%净资产，管理费≤12%总支出
  800万~6000万 → 支出≥6%，管理费≤13%
  400万~800万 → 支出≥7%，管理费≤15%
  <400万 → 支出≥8%，管理费≤20%

年度管理费用<20万 → 不受比例限制
```

---

## 九、违规后果（第十五条）

违反本规定的，由民政部门依法给予**行政处罚**，并通报财政、税务等有关部门。
