# 公益事业捐赠票据使用管理办法 核心要点摘要

> 来源：《公益事业捐赠票据使用管理办法》（财综〔2024〕1号）
> 施行日期：2024年2月1日
> 本文件为法规核心内容的结构化摘要，供 AI 辅助工作时参考。完整原文请查阅财政部官网或中国政府网。

---

## 一、基本定义

- **捐赠票据**：公益性单位依法接受捐赠财物时，向捐赠人开具的凭证
- **适用主体**：县级以上人民政府及其部门、公益性事业单位、公益性社会组织
- **法律效力**：电子票据和纸质票据具有同等法律效力
- **重要用途**：可作为捐赠人申请税前扣除的有效凭证

## 二、票据内容要素

票据基本内容包括：
- 票据名称、票据监制章、票据代码、票据号码
- 交款人统一社会信用代码、交款人名称
- 校验码、开票日期、二维码/条形码
- 项目编码、项目名称、单位、数量、标准
- 金额（大写和小写）、备注
- 收款单位（章）、复核人、收款人

纸质票据联次：存根联（开票方留存）、收据联（捐赠人收执）、记账联（开票方记账凭证）

## 三、必须开具票据的情形

1. 县级以上人民政府及其部门接受的捐赠
2. 公益性事业单位接受的公益捐赠
3. 公益性社会组织接受的公益捐赠
4. 财政部门认定的其他情形

## 四、禁止使用票据的情形

以下情形 **不得** 使用公益事业捐赠票据：
- 以捐赠名义从事营利活动
- 与出资人利益相关的行为
- 集资、摊派、筹资、赞助等行为

## 五、票据领用管理

| 规则 | 要求 |
|------|------|
| 领用凭证 | 须持有《财政票据领用证》 |
| 领用额度 | 一次不超过6个月使用量 |
| 领用制度 | 凭证领用、分次限量、核旧领新 |
| 首次领用 | 须提供领用申请、使用范围和项目说明 |
| 再次领用 | 须提交前次核销情况 |

## 六、使用规范（核心）

### 6.1 基本要求
- 按票据号段 **顺序使用**，不得跳号
- 字迹清楚，内容完整、真实，印章齐全
- 各联次内容和金额一致

### 6.2 开具金额
- **货币捐赠**：按实际收到的金额填开
- **非货币捐赠**：按公允价值填开
- **外币捐赠**：按实际收到的金额填开

### 6.3 错误处理
- **电子票据**：开具红字电子票据
- **纸质票据**：另行填写，作废票据加盖"作废"戳记或注明"作废"字样，保存全部联次

### 6.4 严禁行为
- ❌ 转让、出借、代开、买卖票据
- ❌ 擅自销毁票据
- ❌ 涂改票据
- ❌ 将捐赠票据与其他财政票据、发票互相串用

## 七、核销与销毁

| 事项 | 要求 |
|------|------|
| 核销申请 | 向同级财政部门提交使用情况说明（起止号码、使用份数、作废份数、收取金额） |
| 存根保存 | 纸质票据存根保存期限一般为 **5年** |
| 销毁 | 登记造册，报经财政部门核准后由财政部门组织销毁 |
| 遗失处理 | 发现之日起 **3日内** 登报声明作废，书面报送财政部门备案 |

## 八、机构变更时的处理

撤销、改组、合并的公益性单位，办理《财政票据领用证》变更/注销时：
- 对已使用的纸质票据存根及未使用的票据登记造册
- 交送同级财政部门统一核准、销毁

## 九、电子票据专项

- 财政部门应积极推广电子票据
- 全国统一式样、编码规则和数据标准
- 可通过全国财政电子票据查验平台查验真伪
- 使用单位应准确、完整、有效接收和读取电子票据
