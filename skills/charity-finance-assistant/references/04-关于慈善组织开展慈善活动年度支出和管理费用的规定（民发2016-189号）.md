# 关于慈善组织开展慈善活动年度支出和管理费用的规定

> **文号**：民发〔2016〕189号
> **发文机关**：民政部、财政部、国家税务总局
> **成文日期**：2016年10月11日
> **来源**：中国政府网 https://www.gov.cn/zhengce/zhengceku/2016-10/10/content_5554651.htm

---

## 第一条（立法目的）

为进一步明确慈善组织开展慈善活动的年度支出和管理费用，根据《中华人民共和国慈善法》的有关要求，制定本规定。

## 第二条（基本原则）

慈善组织应当依照法律法规和本组织章程的规定积极开展慈善活动，充分、高效运用慈善财产，并遵循**管理费用最必要原则**，厉行节约，减少不必要的开支。

## 第三条（会计核算）

慈善组织应当依据《民间非营利组织会计制度》，加强对慈善活动相关费用的会计核算。

## 第四条（慈善活动支出的定义）

慈善活动支出是指慈善组织基于慈善宗旨，在章程规定的业务范围内开展慈善活动，向受益人捐赠财产或提供无偿服务时发生的下列费用：

1. 直接或委托其他组织资助给受益人的款物；
2. 为提供慈善服务和实施慈善项目发生的**人员报酬、志愿者补贴和保险**，以及使用房屋、设备、物资发生的相关费用；
3. 为管理慈善项目发生的**差旅、物流、交通、会议、培训、审计、评估**等费用。

> 慈善活动支出在"业务活动成本"项目下核算和归集。

## 第五条（管理费用的定义）

管理费用是指按照《民间非营利组织会计制度》规定，为保证本组织正常运转所发生的下列费用：

1. 理事会等决策机构的工作经费；
2. **行政管理人员**的工资、奖金、住房公积金、住房补贴、社会保障费；
3. 办公费、水电费、邮电费、物业管理费、差旅费、折旧费、修理费、租赁费、无形资产摊销费、资产盘亏损失、资产减值损失、因预计负债所产生的损失、聘请中介机构费等。

## 第六条（共同费用分配）

某些费用如果属于慈善活动、其他业务活动、管理活动共同发生，且不能直接归属某一类活动的，应当将这些费用按照**合理的方法**在各项活动中进行分配。

## 第七条（具有公开募捐资格的慈善组织）

| 组织类型 | 年度慈善活动支出 | 年度管理费用 |
|---|---|---|
| 具有公开募捐资格的**基金会** | ≥ 上年总收入的 **70%** | ≤ 当年总支出的 **10%** |
| 具有公开募捐资格的**社会团体和社会服务机构** | ≥ 上年总收入的 **70%** | ≤ 当年总支出的 **13%** |

## 第八条（不具有公开募捐资格的基金会）

| 上年末净资产 | 年度慈善活动支出 | 年度管理费用 |
|---|---|---|
| ≥ 6000万元 | ≥ 上年末净资产的 **6%** | ≤ 当年总支出的 **12%** |
| 800万元（含）～6000万元 | ≥ 上年末净资产的 **6%** | ≤ 当年总支出的 **13%** |
| 400万元（含）～800万元 | ≥ 上年末净资产的 **7%** | ≤ 当年总支出的 **15%** |
| < 400万元 | ≥ 上年末净资产的 **8%** | ≤ 当年总支出的 **20%** |

## 第九条（不具有公开募捐资格的社会团体和社会服务机构）

| 上年末净资产 | 年度慈善活动支出 | 年度管理费用 |
|---|---|---|
| ≥ 1000万元 | ≥ 上年末净资产的 **6%** | ≤ 当年总支出的 **13%** |
| 500万元（含）～1000万元 | ≥ 上年末净资产的 **7%** | ≤ 当年总支出的 **14%** |
| 100万元（含）～500万元 | ≥ 上年末净资产的 **8%** | ≤ 当年总支出的 **15%** |
| < 100万元 | ≥ 上年末净资产的 **8%** 且 ≥ 上年总收入的 **50%** | ≤ 当年总支出的 **20%** |

> ⚠️ 最后一档（<100万净资产）有**双重标准**，需同时满足。

## 第十条（计算规则的灵活规定）

- 可以用**前三年收入平均数**代替上年总收入
- 可以用**前三年年末净资产平均数**代替上年末净资产
- 上年总收入 = 上年实际收入 - 上年收入中时间限定为上年不得使用的限定性收入 + 于上年解除时间限定的净资产

## 第十一条（小额管理费用豁免）

> **年度管理费用低于20万元人民币的慈善组织，不受第七条、第八条、第九条规定的年度管理费用比例的限制。**

## 第十二条（特殊情形报告义务）

因下列情形导致年度管理费用难以符合要求的，应报告并向社会公开说明：
1. 登记或认定为慈善组织未满1年，尚未全面开展慈善活动的；
2. 折旧费、无形资产摊销费、资产盘亏损失、资产减值损失突发性增长的；
3. 因预计负债所产生的损失突发性增长的。

## 第十三条（捐赠协议约定）

慈善组织签订捐赠协议对单项捐赠财产的支出和管理费用有约定的，从其约定。

## 第十四条（信息公开）

慈善活动支出和管理费用应当在年度工作报告中进行详细披露，并依法向社会公开。

## 第十五条（法律责任）

违反本规定要求的，由民政部门依法给予**行政处罚**并通报财政、税务等有关部门。
