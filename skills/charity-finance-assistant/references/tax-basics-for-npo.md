# 社会组织税务基础知识

> 本文件面向公益机构一线财务人员，用简明语言介绍社会组织涉及的主要税种和免税政策。
> 具体税务处理请以主管税务机关意见为准，建议咨询专业税务师。

---

## 一、核心认知：公益组织不等于不用交税

很多一线人员以为"我们是公益组织就不用交税"——**这是误解**。

正确理解：
- 社会组织是独立的**纳税主体**，依法需要进行税务登记和申报
- 符合条件的可以申请**免税资格**，部分收入可免征企业所得税
- 但不是所有收入都免税，也不是所有税种都免

---

## 二、社会组织涉及的主要税种

| 税种 | 是否涉及 | 说明 |
|------|---------|------|
| **企业所得税** | ✅ | 最重要！取得免税资格后捐赠收入等可免 |
| **增值税** | ✅ | 提供服务/培训收费可能涉及；捐赠收入不涉及 |
| **个人所得税** | ✅ | 为员工代扣代缴工资个税；劳务报酬个税 |
| **印花税** | ⚡ | 签订合同时可能涉及 |
| **房产税/城镇土地使用税** | ⚡ | 自有房产/土地时涉及，部分可申请减免 |
| **车船税** | ⚡ | 有机构车辆时涉及 |
| **契税** | ❌ | 接受房产捐赠时一般免征 |

## 三、非营利组织免税资格（重中之重）

### 3.1 什么是免税资格？

取得免税资格后，以下收入可以免征 **企业所得税**：
1. **捐赠收入**
2. **会费收入**（社会团体）
3. **政府补助收入**（但政府购买服务收入需注意区分）
4. **不征税收入和免税收入孳生的银行存款利息收入**
5. 财政部、国家税务总局规定的其他免税收入

### 3.2 免税资格认定条件

依据《关于非营利组织免税资格认定管理有关问题的通知》（财税〔2018〕13号），需同时满足：

1. 依照国家法律法规设立或登记，具有法人资格
2. 从事公益性或非营利性活动
3. 取得的收入除用于合理支出外，全部用于登记核定的公益性/非营利性事业
4. 财产及其孳息不用于分配（不含合理工资薪金支出）
5. 投入人对投入该组织的财产不保留或享有任何财产权利
6. 工作人员工资福利开支控制在规定的比例内（不超过税务登记所在地同行业同类组织平均工资水平的两倍）
7. 注销后剩余财产用于公益性/非营利性目的
8. 申请前年度的检查结论为"合格"
9. 申请前年度未发生逃避缴纳税款等违法行为

### 3.3 申请流程

```
1. 向主管税务机关提交申请材料
   - 申请报告
   - 事业单位法人证书/社会团体法人登记证书/民办非企业单位登记证书 复印件
   - 章程
   - 上一年度财务报表和审计报告
   - 登记管理机关出具的事业单位、社会团体、基金会年度检查合格证明

2. 省级以上财政、税务部门联合审核

3. 公布名单，资格有效期为 **5年**
4. 到期前 **6个月内** 提出复审申请，否则资格自动失效
```

> ⚠️ **容易混淆的两个资格**：
> | | 非营利组织免税资格 | 公益性捐赠税前扣除资格 |
> |---|---|---|
> | **法规依据** | 财税〔2018〕13号 | 2020年第27号公告 |
> | **有效期** | **5年** | **3年** |
> | **影响谁** | 机构自身（收入是否免企业所得税） | 捐赠人（能否凭票据税前扣除） |
> | **审批部门** | 财政+税务 | 财政+税务+民政 |
> 两者需要**分别申请**，拥有一个不等于拥有另一个！

### 3.4 注意事项
- 免税资格需要**主动申请**，不会自动获得
- 免的是**企业所得税中的部分收入**，不是全部免税
- 取得免税资格后仍需按规定**申报纳税**（申报时填免税收入）
- 免税资格有效期内须持续满足条件，每年配合检查

## 四、免税收入 vs 应税收入

| 收入类型 | 是否免企业所得税 | 是否涉及增值税 | 说明 |
|---------|---------------|-------------|------|
| 个人/企业捐赠 | ✅ 免（有免税资格） | ❌ 不涉及 | 最核心的免税收入 |
| 政府无偿拨款 | ✅ 免 | ❌ 不涉及 | 财政拨款性质 |
| 会费收入 | ✅ 免 | ❌ 不涉及 | 社会团体 |
| 银行存款利息 | ✅ 免 | ❌ 不涉及 | 免税收入孳生的利息 |
| **政府购买服务** | ⚠️ 需区分 | ⚠️ 可能涉及 | 有对价的服务收入可能应税 |
| **培训收费** | ❌ 应税 | ✅ 涉及 | 属于提供服务收入 |
| **咨询服务费** | ❌ 应税 | ✅ 涉及 | 属于提供服务收入 |
| **出租场地/设备** | ❌ 应税 | ✅ 涉及 | 属于租赁收入 |
| 投资收益 | ⚠️ 需区分 | ❌ 不涉及 | 看是否属于免税收入孳生 |

> ⚠️ **特别提醒**：政府购买服务的收入，如果有明确的对价关系（政府付钱、机构提供服务），可能被认定为应税收入。建议咨询主管税务机关。

## 五、个人所得税代扣代缴

公益机构作为用人单位，发放以下款项时须代扣代缴个税：

| 支付项目 | 是否需代扣个税 | 适用税目 |
|---------|-------------|---------|
| 员工工资 | ✅ | 综合所得（工资薪金） |
| 外聘讲师/顾问劳务费 | ✅ | 劳务报酬所得 |
| 志愿者小额补贴 | ⚡ 看金额和性质 | 低于免征额的可能不涉及 |
| 受益人救助金 | ❌ | 通常不属于个税应税范围 |
| 理事/监事津贴 | ✅ | 劳务报酬所得 |

## 六、增值税基础

### 6.1 社会组织是否需要缴增值税？

- **捐赠收入**：不属于增值税应税范围，不缴
- **提供服务收费**（培训、咨询等）：属于增值税应税行为
- **小规模纳税人**：大部分社会组织为小规模纳税人，适用简易征收

### 6.2 小规模纳税人优惠

- 月销售额≤10万元（季度≤30万元）：免征增值税
- 适用征收率通常为3%（部分时期有减按1%的优惠政策）

> 💡 很多小机构因为服务收入不高，实际享受免征优惠。但仍需按期申报。

### 6.3 增值税计算示例

**示例一：季度服务收入≤30万（免征）**

```
某基金会本季度开展了2期收费培训：
  - 第1期培训收费：8万元
  - 第2期培训收费：5万元
  - 季度服务收入合计：13万元

判断：13万 ≤ 30万（季度免征额度）
结果：本季度增值税免征 ✅
操作：在增值税申报表"小微企业免税销售额"栏填 130,000 元
```

**示例二：季度服务收入>30万（需缴税）**

```
某社会组织本季度：
  - 培训收费：25万元
  - 咨询服务费：12万元
  - 季度服务收入合计：37万元

判断：37万 > 30万 → 需缴纳增值税
税率：小规模纳税人征收率 3%
应缴增值税 = 370,000 ÷ (1+3%) × 3% = 10,776.70 元

注意：
  - 37万是含税收入，需先换算为不含税收入再计税
  - 不含税收入 = 含税收入 ÷ (1+征收率)
  - 捐赠收入不计入增值税应税收入
```

**示例三：混合收入的区分**

```
某基金会本季度收入：
  - 捐赠收入：200万元 → 不涉及增值税
  - 政府无偿拨款：50万元 → 不涉及增值税
  - 培训收费：8万元 → 增值税应税
  - 咨询服务：3万元 → 增值税应税
  - 银行利息：0.5万元 → 不涉及增值税

增值税应税收入合计 = 8 + 3 = 11万元
判断：11万 ≤ 30万 → 免征 ✅

关键点：判断是否超30万只看"应税收入"，捐赠收入不算！
```

### 6.4 增值税申报操作指引

**申报周期**：
- 小规模纳税人通常按 **季度** 申报（也可选择按月）
- 申报截止：季度结束后次月15日前（如1月/4月/7月/10月的15日前）

**申报方式**：
- 通过**电子税务局**网上申报（推荐）
- 或到主管税务机关办税服务厅现场申报

**填报要点（小规模纳税人）**：

| 申报表栏目 | 填什么 | 说明 |
|-----------|-------|------|
| 应征增值税不含税销售额 | 超过30万的季度填此栏 | 不含税金额 |
| 小微企业免税销售额 | 未超30万的季度填此栏 | 含税金额 |
| 免税销售额 | 不涉及增值税的收入不需填 | 捐赠收入不在此表体现 |

**零申报**：
- 如本季度没有任何增值税应税收入，仍需进行**零申报**
- 在电子税务局上全部填0提交即可

> ⚠️ **常见错误**：把捐赠收入填到增值税申报表里。捐赠收入不属于增值税应税范围，不需要在增值税申报表中体现。

## 七、常见税务问题速答

**Q：我们是公益组织，还需要做税务登记吗？**
> 需要。社会组织取得登记证书后30日内须向主管税务机关办理税务登记。

**Q：每月/每季都要报税吗？**
> 是的。即使收入全部免税，也需按期进行纳税申报（零申报或填报免税收入）。

**Q：捐赠的物资也要交税吗？**
> 接受捐赠方一般不因收到捐赠而产生纳税义务。但捐赠方（企业）视同销售可能涉及增值税。

**Q：我们办了一个收费培训班，收入要交税吗？**
> 培训收费属于应税收入，涉及增值税和企业所得税。如为小规模纳税人且季度收入≤30万，增值税可免。企业所得税需在汇算清缴时缴纳（如有免税资格，此部分收入不免）。
