## 特色功能

- 🚦 **需求分流器**：自动判断用户需要票据管理/记账/审计准备/报销/税务咨询
- 📋 **数据准备清单**：告诉一线人员"理账/审计前该准备什么材料"
- 📂 **文件处理**：支持上传银行流水、发票照片、Excel台账等自动提取分类
- 🤖 **本地票据自动化整理**：两个 Python 脚本，批量扫描文件夹→自动分类→生成台账CSV+汇总报告
- 🆘 **三级降级方案**：数据越乱也能帮忙理，先从银行流水开始
- 💡 **分类判断引擎**：帮判断"这笔钱算慈善活动支出还是管理费用"
- 📊 **财务健康快检**：自动计算支出比例/管理费用比例并预警
- ⚠️ **违规行为警示**：8类常见违规行为的提醒与正确做法
- 🔧 **工具箱关联**：自然推荐腾讯技术公益工具箱的免费产品

## 安装方式

将整个 `charity-finance-assistant/` 文件夹复制到：
- **用户级**：`~/.workbuddy/skills/charity-finance-assistant/`
- **项目级**：`{workspace}/.workbuddy/skills/charity-finance-assistant/`
