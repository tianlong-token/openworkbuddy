# 支付宝/蚂蚁公益 — 平台格式规范

## 项目上线申请

### 必填模块

1. **基础信息**：项目名称、项目类型（短期/长期）、筹款目标金额、筹款周期/执行期限、目标地区、受益人数、服务领域/分类
2. **项目文案（核心）**：项目简介/概要 + 项目详情文案。详情文案必须包含完整链路：
   - 项目背景
   - 项目目标
   - 实施计划
   - 预算方案
   - 执行安排
   - 捐赠票据获取方式
3. **执行能力证明**：执行机构介绍、过往案例/成效（可选但建议提供）
4. **图片物料**：项目封面/头图、项目详情配图、机构资质证照

### 关键注意事项

- **高质量项目评审制**：平台对文案逻辑、预算合理性、合规性要求较高
- **完整链路必须齐**：背景 → 目标 → 计划 → 预算 → 执行 → 票据方式，缺任何一段都容易被打回
- **执行时效硬约束**：短期项目筹满后通常需在 12 个月内完成执行并提交结项报告；长期项目也需关注年度执行率
- **非公募机构路径**：可与具备公募资格的合作基金会确认挂靠流程
- **票据说明要明确**：捐赠人如何获取票据/收据，最好写成单独小节

## 中期/结项报告

建议重点覆盖：项目基本信息、执行进展、受益人数变化、预算执行情况、票据与信息公开说明、后续计划/结项说明。

## 📌 平台操作指引

- **平台入口**：https://opendocs.alipay.com/b/03ah6m（项目发布要求页）
- **常用路径**：注册企业支付宝账号 → 完成实名认证 → 申请入驻公益板块 → 审核通过后发起项目
- **操作建议**：提交前重点复核项目详情文案是否已写完整链路；文案逻辑和标点细节都会被审查
- **补充说明**：如平台要求线上答辩/评审，文书中的预算和执行安排要与答辩口径保持一致
