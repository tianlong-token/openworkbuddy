# 通用平台格式规范

## 其他/未列出平台的通用项目上线结构

当用户选择垂直筹款平台、基金会/枢纽平台、地方慈善会平台，或提供了未内置的平台名称时，默认按以下结构起草，再结合用户补充的平台要求微调：

1. **基础信息**：项目名称、执行主体、公募合作方（如有）、项目周期、实施地区、筹款/预算目标、受益人数
2. **项目背景**：问题现状、目标群体画像、为什么需要此项目、已有资源基础
3. **项目目标与方案**：总目标、分目标、核心服务内容、执行步骤、时间安排
4. **预算与资金使用**：预算明细、资金用途、管理费用说明、票据/收据获取方式（如平台有要求）
5. **执行保障**：执行团队、合作伙伴、风险控制、监测评估方式
6. **传播与反馈**：传播计划、信息公开、捐赠人反馈、结项交付
7. **附件物料**：封面图、项目配图、机构资质、授权书、预算表、补充说明

### 分类使用提示

- **垂直筹款平台**：优先写清个案信息、筹款用途、透明反馈机制
- **基金会/枢纽平台**：优先写清项目逻辑、预算合理性、执行能力、评估指标与附件
- **其他/未列出平台**：告诉用户可先按通用公益规范生成，再根据平台特点二次收口

## 肖像及资料授权书

### 必备要素

| 要素 | 内容 |
|------|------|
| 授权方 | 姓名、身份证号、联系电话 |
| 被授权方 | 机构全称、地址 |
| 授权背景 | 关联项目名称 |
| 授权内容 | 肖像、姓名、基本信息 |
| 使用范围 | 全球全媒体，**仅限公益用途** |
| 修改限制 | 不得歪曲、篡改 |
| 期限 | 永久（或约定期限） |
| 签署 | 签字 + 手印 + 日期 |
| 附件 | 授权使用的照片 |

### ⚠️ 未成年人特别条款

在成年人版基础上**必须额外包含**：
- 监护人声明（确认为合法监护人）
- 监护人信息（姓名、身份证号）
- 监护人签署（代替未成年人签字）
