# 合规知识速查与自查清单

> 生成结项报告/年报类文书时必须参考本文件。

---

## 一、合规硬性指标

### 慈善活动支出（最低比例）

| 类型 | 标准 |
|------|------|
| 公募基金会 | ≥ 上年总收入的 **70%** |
| 非公募基金会 | ≥ 上年末净资产的 **8%** |

### 管理费用（上限比例）

| 年度慈善活动支出 | 管理费用上限 |
|----------------|------------|
| ≤ 1000万元 | ≤ 当年总支出的 **10%** |
| 1000万 ~ 1亿元 | ≤ **12%** |
| > 1亿元 | ≤ **15%** |

### 信息公开关键时间节点

| 截止日期 | 事项 |
|---------|------|
| 每年3月31日前 | 向民政部报送年度工作报告 |
| 每年5月31日前 | 在"慈善中国"公开年报和财报 |
| 募捐结束后3个月内 | 公开募得款物和使用情况 |
| 项目结束后3个月内 | 公开项目实施情况 |

---

## 二、生成后自查要点

### 通用必查项
- 所有金额数字在不同章节是否一致
- 受益人信息是否已脱敏（姓名/身份证/住址）
- 附件清单是否完整

### 结项报告专项
- 善款明细 **支出合计 = 实际总支出 + 公募管理费**
- 票据日期在项目上线时间**之后**
- 执行照片描述 ≥ 5张

### 年度报告专项
- 慈善活动支出占比是否达标（见上方比例表）
- 管理费用占比是否在限额内
- 主要捐赠人（超5%或500万）是否已列出
- 关联交易是否已说明
- 提醒用户准备法定代表人和监事签名 + 组织印章

### 年检常见退回原因

| 退回原因 | 如何避免 |
|---------|---------|
| 财务数据不一致 | 以审计报告数据为准，逐项核对 |
| 支出比例不达标 | 生成报告时自动计算并预警 |
| 管理费用超标 | 生成报告时自动计算并预警 |
| 信息不完整 | 对照平台字段逐项检查 |
| 签章不全 | 在报告末尾提醒签章要求 |
