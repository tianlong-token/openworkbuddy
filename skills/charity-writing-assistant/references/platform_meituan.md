# 美团公益 — 平台格式规范

## 项目上线申请

### 必填模块

1. **基础信息**：项目名称、发起机构信息、项目领域/分类、筹款目标、执行周期、目标地区、受益人数
2. **项目详情**：项目背景与问题分析、解决方案/实施内容、预算方案、预期效果与指标
3. **执行保障**：执行团队能力、风险控制措施（可选但建议补齐）
4. **物料**：项目封面/图片、机构资质文件

### 关键注意事项

- **本地化倾向明显**：适合社区服务、城市服务、乡村振兴、应急救助等与本地生活网络相关的项目
- **合作公募要求要问清**：平台公募项目通常需要具备公募资质的合作机构参与
- **强调项目管理**：建议在文书里写清项目管理流程、监测评估与风险控制
- **企业/员工公益联动**：若项目可接入企业 CSR 或员工志愿者机制，可在方案中主动体现

## 阶段/结项报告

建议重点覆盖：项目基本信息、执行进展、受益人数与变化、预算执行、量化指标达成情况、团队复盘与风险应对、下一步计划。

## 📌 平台操作指引

- **平台入口**：https://gongyi.meituan.com
- **常用路径**：查看入驻协议 → 提交机构入驻申请 → 审核通过后在后台发起项目 → 填写并提交审核
- **操作建议**：若项目偏社区/城市服务，文书中要强化本地场景、服务半径、合作社区/街道等信息
- **补充说明**：上线后可结合美团 App、本地生活场景、员工公益资源进行传播
