# Cron 表达式参考

格式：`秒 分 时 日 月 周`（秒固定为 0）

| 位置 | 字段 | 取值 |
|------|------|------|
| 1 | 秒 | 固定 0 |
| 2 | 分 | 0-59 |
| 3 | 时 | 0-23 |
| 4 | 日 | 1-31 |
| 5 | 月 | 1-12 |
| 6 | 周 | 0-6 (0=周日) |

特殊字符：`*` 任意值、`,` 列表（如 `1,3,5`）、`-` 范围（如 `1-5`）、`/` 步进（如 `*/5`）

## 核心示例

| 描述 | 表达式 |
|------|--------|
| 每分钟 | `0 * * * * *` |
| 每5分钟 | `0 */5 * * * *` |
| 每30分钟 | `0 */30 * * * *` |
| 每小时 | `0 0 * * * *` |
| 每2小时 | `0 0 */2 * * *` |
| 每天 09:00 | `0 0 9 * * *` |
| 每天 9:00 和 18:00 | `0 0 9,18 * * *` |
| 工作时间每小时 (9-17) | `0 0 9-17 * * *` |
| 工作日 09:00 | `0 0 9 * * 1-5` |
| 周末 10:00 | `0 0 10 * * 0,6` |
| 每周一 09:00 | `0 0 9 * * 1` |
| 每月1号 09:00 | `0 0 9 1 * *` |
| 每季度第一天 09:00 | `0 0 9 1 1,4,7,10 *` |

## 自然语言 → Cron

| 用户说 | 表达式 |
|--------|--------|
| "每天早上9点" | `0 0 9 * * *` |
| "每天下午3点" | `0 0 15 * * *` |
| "每小时" | `0 0 * * * *` |
| "每30分钟" | `0 */30 * * * *` |
| "每周一上午10点" | `0 0 10 * * 1` |
| "工作日早上9点" | `0 0 9 * * 1-5` |
| "每月1号" | `0 0 9 1 * *` |
| "每两小时" | `0 0 */2 * * *` |
